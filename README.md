# 🌟 Questly – Kids Adventure Planner (Ages 4–12)

A fun, colorful, gamified monthly planner + daily to-do list + mental wellness journal made for kids.

### ✨ Features
- Interactive onboarding (name, age, interests)
- Monthly Big Dreams that turn into Daily Quests
- Colorful quest cards with confetti rewards + Adventure Coins
- 45-minute focus timer + creative break suggestions (painting, dancing, etc.)
- End-of-day journaling + gratitude space
- Streak counter and offline reminders
- 100% works offline

### 🛠️ Built With
- HTML + Tailwind CSS + Vanilla JavaScript
- Data saved in browser localStorage
